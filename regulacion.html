Como cumplir con la regulacion

Cumplir con las normativas de las distintas entidades regulatorias es sumamente importante no solo para evitar sanciones o inclusive la suspensión de las operaciones, sino también para alinearse a las mejores prácticas de mercado, para así, disminuir riesgos operativos que permitan tener una operación sustentable a largo plazo.

Seguridad

Toda la información que se trasmite o almacena en Quanto® es encriptada y administrada bajo los estándares mas altos de seguridad del Sector Financiero, cuando un usuario inicia sesión y opera toda la información que se solicita al servidor es encriptada mediante SSL, adicionalmente la información que es almacenada en la base de datos encriptada para que no pueda ser extraída de manera indebida.

Actualización regulatoria sin costo

Quanto® está desarrollado con tecnología de punta bajo estándares internacionales.

Cumplimiento

Quanto® ofrece una gran cantidad de elementos para dar cumplimiento a la normativa de SHCP, SAT, CNBV, CNSF, Condusef, entre otros. Tanto para Arrendadoras y empresas de Factoraje, SOFOMES, SOFIPOS, SOCAPs, Uniones de Crédito, Fintech y Bancos.

Algunos de estos elementos incluyen Matrices de Riesgo (EBR), Conocimiento del Cliente (KYC), Alertas automáticas, limites de operación. Asi como reportes que se envían a las entidades regulatorias como reportes de operaciones relevantes, inusuales e internas preocupantes

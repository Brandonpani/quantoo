Plataforma

Quanto® es una plataforma desarrollada con tecnología de punta, que se hospeda en la nube de Microsoft Azure con la capacidad de escalar de manera automática el numero de procesadores, espacio de memoria y espacio en disco duro, por lo cual ofrece un tiempo promedio de respuesta menor a 700 milisegundos por solicitud.

Explora la plataforma

Seguridad

Quanto® se hospeda en centros de datos de Microsoft Azure que cumplen con estándares internacionales, por lo cual su información siempre estará segura y disponible.

Estándares Internacionales

Microsoft Azure cumple un conjunto amplio de estándares de cumplimiento internacionales y específicos de la industria, como ISO 27001, HIPAA, FedRAMP, SOC 1 y SOC 2.

Encriptación

Para la comunicación se utiliza encriptación de comunicación mediante certificado SSL que permite encriptar la información que viaja entre el servidor y el equipo de usuario final. Para los datos almacenados se utiliza cifrado de datos transparente (TDE) protege los datos y ayuda a cumplir los requisitos de cumplimiento cifrando la base de datos, las copias de seguridad asociadas y los archivos de registro de transacciones en reposo.

Arquitectura

La funcionalidad de la solución esta distribuida en capas que se implementan en servidores por separado, lo que permite implementar los componentes que no requieren autenticación en zonas públicas y los demás en zonas privadas.

Administración de Amenazas

Quanto® cuenta con detección de intrusiones, prevención de ataques de denegación de servicio (DDoS), pruebas de penetración regulares, análisis de datos y herramientas de aprendizaje automático para mitigar las amenazas a la plataforma.

Una solución con tecnología de punta.

Quanto® está desarrollado con tecnología de punta bajo estándares internacionales.

Rendimiento

Quanto® utiliza tecnología de punta para brindar un alto desempeño ofreciendo al usuario final respuesta inmediata, con un tiempo promedio de respuesta menor a 700 milisegundos por solicitud.

Preparese para el Futuro

Quanto® le permite prepararse para el futuro de manera fácil y rápida, ya que con Quanto no requiere cambiar de software o hardware si el volumen de transacciones aumenta, ya que tiene la capacidad de escalar de manera automática el numero de procesadores, espacio de memoria y espacio en disco duro.

Escalado Automático

La escalabilidad automática ayuda a rendir de forma óptima cuando cambia la demanda. Por ejemplo, puede que atienda millones de solicitudes en dias hábiles pero ninguna durante el fin de semana, el escalado automático permite escalar el servicio en función de estas circunstancias o de cualquier otra métrica que defina.

Rendimiento de Base de Datos

Quanto® utiliza el servicio Database Advisor que valúan y analizan continuamente el historial de uso con Machine Learning, para proporcionar recomendaciones basadas en patrones de carga de trabajo que ayudan a mejorar el rendimiento.

Rendimiento de aplicación

Quanto® utiliza el servicio Application Insights que supervisa la aplicación web. Este permite detectar y diagnosticar problemas de rendimiento en tiempo real.

En la Nube

Quanto® opera en la nube lo que permite eliminar todos los costos relacionados al hardware del Core Financiero. Con Quanto® opere con tecnología de punta de la mano de expertos cumpliendo con estándares Internacionales como PCI DSS e ISO 27001.

Disminuya costos

Quanto® opera en la nube de Microsoft Azure lo que permite eliminar todos los costos relacionados al hardware del Core Financiero, así como los costos relacionados al personal técnico especializado para su instalación y mantenimiento.

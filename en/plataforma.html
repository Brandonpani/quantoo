Platform

Quanto® is a platform developed with state-of-the-art technology, which is hosted in the Microsoft Azure cloud with the ability to automatically scale the number of processors, memory space and hard disk space, for which it offers an average time of response less than 700 milliseconds per request.

Explore the platform

Security

Quanto® is hosted in Microsoft Azure data centers that meet international standards, so your information will always be safe and available.

International Standards

Microsoft Azure complies with a broad set of industry-specific and international compliance standards, such as ISO 27001, HIPAA, FedRAMP, SOC 1, and SOC 2.

Encryption

For communication, communication encryption is used through an SSL certificate that allows encrypting the information that travels between the server and the end user's equipment. Transparent Data Encryption (TDE) is used for stored data to protect data and help meet compliance requirements by encrypting the database, associated backups, and transaction log files at rest.

Architecture

The functionality of the solution is distributed in layers that are implemented on separate servers, which allows the components that do not require authentication to be implemented in public zones and the others in private zones.

Threat Management

Quanto® features intrusion detection, Denial of Service (DDoS) prevention, regular penetration testing, data analytics, and machine learning tools to mitigate threats to the platform.

Performance

Quanto® uses state-of-the-art technology to provide high performance, offering the end user an immediate response, with an average response time of less than 700 milliseconds per request.

Prepare for the Future

Quanto® allows you to prepare for the future easily and quickly, since with Quanto you do not need to change software or hardware if the volume of transactions increases, since it has the ability to automatically scale the number of processors, memory space and hard drive space.

Automatic Scaling

Automatic scalability helps you perform optimally when demand changes. For example, you may serve millions of requests on business days but none on the weekends, autoscaling allows you to scale the service based on these circumstances or any other metric you define.

Database Performance

Quanto® uses Database Advisor services that continuously assess and analyze usage history with Machine Learning to provide recommendations based on workload patterns that help improve performance.

Application performance

Quanto® uses the Application Insights service that monitors the web application. This allows you to detect and diagnose performance problems in real time.

On the cloud

Quanto® operates in the cloud, which allows you to eliminate all the costs related to the hardware of the Financial Core. With Quanto®, operate with state-of-the-art technology hand in hand with experts, complying with international standards such as PCI DSS and ISO 27001.

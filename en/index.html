Security controls

Quanto® allows you to establish security controls required by regulatory entities. Such as user profiles, query levels, centralized logs and rules for the generation and management of secure passwords.

Information security

Quanto® has services, policies and practices that allow you to encrypt data in motion and stored it, automatic backups, redundancy models, secure data deletion and vulnerability detection.

Compliance

It integrates Anti Money Laundery module with functions such as risk matrix editor (EBR), customer profiling (KYC) and alert generation, as well as Portfolio Rating and Regulatory Report modules.

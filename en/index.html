Security standards aligned to the financial sector.

Quanto® allows establishing efficient security controls, as well as providing services, policies and procedures for information security in compliance with regulatory entities. It also integrates modules for Prevention of Money Laundering, Portfolio Rating and Regulatory Reports.

Find out how to comply with the regulation

Security controls

Quanto® allows you to establish the security controls required by regulatory entities. Such as user profiles, query levels, centralized logs and rules for the generation and management of secure passwords.

Security of the information

Quanto® has services, policies and practices that allow the encryption of data in motion and stored, automatic backups, redundancy models, secure data deletion and vulnerability detection.

Compliance

It integrates the Money Laundering Prevention module with functions such as risk matrix editor (EBR), customer profiling (KYC) and alert generation, as well as Portfolio Rating and Regulatory Report modules.
